# Projet Node 2026

## Bot Discord TS website monitoring

### Fonctionnalités principales 
- register (url du serveur et alias) -> uptime configurable
- delete -> supprime l'enregistrement du site
- status (alias) -> renvoie le statut
- list -> voir tous les sites register

### Consignes 
- Doit être fait en TS
- stack -> libre

### Setup
- npm i
- remettre le .env
- docker compose up -d
- npx drizzle-kit push
